# All about VIM text editor

Vim word shortcuts
| Command | Description |
| - | - |
| `x` | Cut character |
| `dw` | Cut word |
| `dd` | Cut full line |
| `yw` | Copy word |
| `yy` | Copy full line |
| `p` | Paste |

we can write a number before any command to, for example copy 4 word we would write `4yw`

Vim commands
| Command | Description |
| :1 | Go to line 1 |
| :w | Write the file, save |
| :q | Quit |
| :q! | Quit without saving |
| wq | Write and quit |

[More handy short-cuts](https://vimsheet.com/)
