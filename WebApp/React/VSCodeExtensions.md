## Extensions

These are personal, but I've noticed they increase productivity, less time in menus, faster text selection, greps

### Visuals
- Color highlight
- escook-theme
- Material icon theme

### QoL
- Auto rename tag
- Bookmarks (meh could help)
- Code spell checker
- Command palette NPM
    - Bind to CTRL + ALT + .
- EOF mark
- Gitlens
- Live server
- Prettier
- Prettier ESLint
- Select by indent
    - Bind to SHIFT + ALT + arrow left or right (based on expand or shrink)
- Stopwatch
    - Bind to SHIFT + CTRL + ALT + (END for stop stopwatch, HOME for reset, R for resume)   
- SVG preview

### React
- HTML CSS support
- i18n ally
- ESlint
- JS and TS nightly
- ES7 + React/Redux
